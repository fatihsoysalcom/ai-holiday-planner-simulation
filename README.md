# AI Holiday Planner Simulation

This example simulates an AI-powered holiday planner. It takes user preferences as input and suggests a personalized holiday plan, demonstrating how AI can tailor recommendations based on user needs without complex machine learning models.

## Language

`python`

## How to Run

Save the code as `main.py`. Open your terminal or command prompt, navigate to the directory where you saved the file, and run it using `python main.py`.
